# Product Overview

AI Try-On is a virtual clothing try-on application that allows users to:

- Upload a full-body photo and virtually try on clothing items using AI
- Build and manage a personal wardrobe/closet of clothing items
- Save, compare, and share outfit combinations
- Browse community-shared outfits with social features (likes, comments, follows)
- Organize clothing into collections

The app targets fashion-conscious users who want to visualize outfits before purchasing or wearing them.

## Key Features

- AI-powered virtual try-on using image generation
- Clothing image analysis and categorization
- Background removal for clothing items
- Multi-language support (Vietnamese, English, Chinese, Korean, Japanese, Thai)
- Social feed with outfit sharing
- User authentication and profiles
